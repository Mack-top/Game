- 自动化构建

- 玩法管理

## Design

现代、简洁、功能性UI，以深色调为主，搭配科技感蓝色和绿色作为点缀色，强调信息的可读性和操作的直观性。采用响应式布局，确保在不同设备上都有良好的用户体验。
